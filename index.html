(V0.6) This tool is used for calculating weight&balance and performance for the DA40. Primarily for use with UVU aircraft. Calculations may not perfectly represent the values derived through use of an approved AFM. The following assumptions are made: wind is 0kts (a headwind will only improve takeoff performance), level-asphalt runway, landing distance is for takeoff weight, V-speeds are calculated using takeoff weight (with the exception of Va which uses landing weight), and oil is at 6qts. For cross country trips, performance should be calculated for both departure and destination.
